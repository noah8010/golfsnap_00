# GolfSnap Mobile - AI 세션 가이드

> **이 파일은 AI 세션 시작 시 반드시 먼저 읽어야 하는 핵심 지침서입니다.**
> 새로운 세션을 시작할 때마다 이 파일을 참조하여 프로젝트 컨텍스트를 파악하세요.

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | GolfSnap Mobile |
| **목적** | 개발자/디자이너에게 전달할 모바일 영상 편집기 프로토타입 |
| **타입** | React 웹 기반 모바일 앱 (모바일 웹뷰 대상) |
| **해상도** | 393x852px (iPhone 14 Pro 기준) |
| **타겟 사용자** | 스크린골프 이용자 |

### 핵심 가치
```
1. 모바일 퍼스트 - 터치 제스처 중심 UX
2. 직관적인 타임라인 편집 - 중앙 고정 플레이헤드
3. 빠른 프로토타이핑 - UI/UX 검증이 목적
```

---

## 2. 기술 스택 (변경 금지)

```
React 18.2 + TypeScript 5.2 + Vite 5.0
Zustand 4.4 (상태 관리)
Tailwind CSS 3.3 (스타일링)
Framer Motion 10.x (애니메이션)
Lucide React (아이콘)
```

## 4. 아키텍처 개요

### 4.1 화면 흐름 (변경 금지)
```
[대시보드] ──새 프로젝트──▶ [비율 선택] ──▶ [미디어 선택] ──▶ [AI 처리] ──▶ [에디터]
    │                                                                        │
    └────────────────────────── 기존 프로젝트 클릭 ──────────────────────────┘
```

### 4.2 핵심 파일 구조
```
src/
├── App.tsx                     # 라우팅 (수정 주의)
├── types/golf.ts               # 타입 정의 (수정 시 전파 영향 큼)
├── store/useAppStore.ts        # 전역 상태 (수정 시 주의)
├── constants/editor.ts         # 상수 값
├── hooks/useTimeline.ts        # 타임라인 핵심 로직 (복잡)
└── screens/EditorWorkspaceScreen.tsx  # 에디터 메인 (가장 복잡)
```

### 4.3 5개 트랙 구조 (변경 금지)
```
┌─────────────────────────────────────────────────┐
│ 영상 (video)        │ 파란색 │ 메인 비디오      │
├─────────────────────────────────────────────────┤
│ 텍스트 (text)       │ 주황색 │ 텍스트 오버레이  │
├─────────────────────────────────────────────────┤
│ 오디오 (audio)      │ 초록색 │ BGM/효과음      │
├─────────────────────────────────────────────────┤
│ 필터 (filter)       │ 보라색 │ 색상 필터       │
├─────────────────────────────────────────────────┤
│ 스티커 (sticker)    │ 핑크색 │ 이모지 스티커   │
└─────────────────────────────────────────────────┘
```

## 7. 절대 규칙 (위반 금지)

### 7.1 수정 금지 항목
```
- 기본 화면 흐름 (대시보드 → 에디터)
- 5개 트랙 구조 및 색상
- 모바일 프레임 크기 (393x852px)
- 중앙 고정 플레이헤드 방식
- 기술 스택 (React, Zustand, Tailwind 등)
```

### 7.2 필수 준수 사항
```
- 모든 컴포넌트는 TypeScript strict 모드
- 모든 함수/컴포넌트에 JSDoc 주석
- Tailwind CSS만 사용 (인라인 스타일 금지)
- Framer Motion으로 애니메이션
- Lucide React 아이콘만 사용
```

### 7.3 코드 품질
```
- 새 기능 추가 전 기존 코드 패턴 확인
- 컴포넌트당 300줄 이하 권장
- 중복 코드 발견 시 즉시 리팩토링
- 타입 any 사용 금지
```
