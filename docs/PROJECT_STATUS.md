|------|------|
| 프로젝트명 | GolfSnap Mobile |

| 해상도 | 393x852 (iPhone 14 Pro) |

## 완성도

### 완료 (100%)

| 기능 | 설명 |
|------|------|
| 프로젝트 대시보드 | 프로젝트 목록, 검색, 정렬 |
| 3단계 새 프로젝트 생성 | 비율 선택 → 미디어 선택 → AI 처리 |
| 5개 트랙 타임라인 | 영상, 텍스트, 오디오, 필터, 스티커 |
| 클립 조작 | 분할, 복제, 삭제, 속도, 트림 |
| 클립 볼륨 조절 | 비디오 클립별 원본 오디오 볼륨/음소거 |
| 클립 오버랩 표시 | 같은 트랙 내 겹침 시각화 (점선 테두리) |
| 클립 수정 기능 | 텍스트/오디오/필터/스티커 클립 수정 (툴바 + 더블클릭) |
| 타임라인 스크롤 동기화 | 좌측 레이블과 우측 트랙 세로 스크롤 동기화 |
| 패널 디자인 통일 | 텍스트/오디오/필터/스티커 패널 전체화면 레이아웃 |
| 중앙 고정 플레이헤드 | 리플 편집 지원 |
| 드래그 & 드롭 | 롱프레스(500ms) 후 이동 |
| 핀치 줌 | 타임라인 확대/축소 (0.5x~3x) |
| 하단 네비게이션 | 균등 분배, 만들기 탭 활성화 |
| 내보내기 플로우 | 설정 → 렌더링(취소 가능) → 완료 후 공유/다운로드/계속편집 |
| 실행 취소/다시 실행 | 히스토리 스택 (최대 30단계), Ctrl+Z/Ctrl+Y |
| 자동 저장 UI | 저장 상태 표시 (저장됨/저장 중/미저장) |
| 텍스트 애니메이션 | 11가지 프리셋 (fade-in, slide-up, glow 등) |
| 클립 스냅 | 드래그 시 다른 클립 경계에 자동 스냅 |
| AI 어시스턴트 | 메타데이터 기반 스티커/텍스트 자동 추천, 메타데이터 없는 미디어는 비활성화 |
| **다중 클립 선택** | 툴바 버튼으로 멀티셀렉트 모드, 일괄 삭제 |
| **미리보기 재생 시뮬레이션** | requestAnimationFrame 기반 플레이헤드 자동 이동 |
| **전환 효과** | 클립 간 None/Fade/Slide/Zoom 전환 설정 |
| **오디오 파형 표시** | Canvas 기반 오디오 트랙 가짜 파형 시각화 |
| **프로젝트 템플릿** | 4가지 프리셋 (기존 프로젝트 생성 흐름 연동) |
| **다크/라이트 모드** | localStorage 기반 테마 전환, 시스템 설정 감지 |
| **에디터 리팩토링** | EditorToolbar, PreviewPlayer, TimelineTrack 컴포넌트 분리 |
| **인앱 토스트 메시지** | useToastStore + Toast 컴포넌트, alert() 전면 제거 |
| **미리보기 필터 피드백** | CSS filter 기반 프리셋/커스텀 필터 실시간 미리보기, 필터 라벨 표시 |
| **하단 네비게이션 전체 활성화** | 5개 탭 모두 활성화, 미확정 화면에 프로토타입 배너 |
| **빈 상태(Empty State) UI** | 타임라인 빈 트랙 힌트, 대시보드 빈 프로젝트 안내 |
