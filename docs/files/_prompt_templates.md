# Claude 프롬프트 템플릿

> **파일**: `_prompt_templates.md`  
> **용도**: 명세서 변경 시 아래 템플릿을 복사하여 Claude에게 전달

---

## 템플릿 1: 사소한 변경 (섹션 1~3개)

```
아래 변경 로그를 반영하여 해당 섹션의 디스크립션만 업데이트해줘.
기존 형식(어노테이션 텍스트 + 조건/상태표)을 유지해줘.

[변경 로그]
(여기에 _changelog.md에서 해당 버전의 변경 내역만 붙여넣기)

[현재 해당 섹션 명세서]
(해당 MD 파일에서 변경된 SECTION만 붙여넣기)
```

---

## 템플릿 2: 중간 규모 변경 (섹션 추가/삭제 포함)

```
아래 변경 로그를 반영해줘. 신규 섹션은 디스크립션을 새로 작성하고,
삭제 섹션은 표시해줘. 노션 DB에 반영할 수 있도록 섹션 ID별로 정리해줘.

[변경 로그]
(여기에 _changelog.md에서 해당 버전 전체 붙여넣기)

[변경된 파일 전체]
(해당 MD 파일 전체 붙여넣기)
```

## 템플릿 3: 새 페이지 추가

```
새 페이지가 추가되었어. 아래 명세서를 기반으로
전체 디스크립션(어노테이션 + 조건/상태표)을 생성해줘.
기존 P1~P6과 동일한 형식으로 만들어줘.

[새 페이지 명세서]
(새 MD 파일 전체 붙여넣기)
```
