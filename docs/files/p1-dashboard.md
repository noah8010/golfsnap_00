| 항목 | 내용 |
|------|------|

### 진입 시 데이터 로딩

| API | 용도 | 로딩 UI |
|-----|------|---------|
| `GET /api/auth/me` | 사용자 정보 (프로필, 테마) | 프로필 영역 스켈레톤 |
| `GET /api/projects?sort=updatedAt:desc&page=1&limit=30` | 프로젝트 목록 | 프로젝트 카드 스켈레톤 |

- 인증 실패(401) 시 → 로그인 화면으로 리다이렉트
- 네트워크 에러 시 → 에러 토스트 + 재시도 버튼

<!-- SECTION: P1-A -->
## A. 헤더

```
┌────────────────────────────────────┐
│ [←]         만들기           [🌙]  │
└────────────────────────────────────┘
```

| 요소 | 동작 |
|------|------|
| ← 뒤로가기 | 홈 화면으로 이동 |
| "만들기" | 고정 텍스트 (페이지 제목) |
| 🌙/☀️ 테마 토글 | 다크모드/라이트모드 전환 |

### 테마 토글

| 항목 | 규칙 |
|------|------|
| 라이트 모드 아이콘 | 🌙 (Moon) — "다크 모드로 전환" |
| 다크 모드 아이콘 | ☀️ (Sun) — "라이트 모드로 전환" |
| 전환 애니메이션 | 아이콘 회전 + 페이드 교체 (200ms) |
| 저장 | `localStorage`에 테마 설정 저장 |
| 초기값 | localStorage → 시스템 설정 → 'light' 순으로 결정 |
| 서버 동기화 | `PATCH /api/users/{id}` (`{ theme: 'dark' | 'light' | 'system' }`) |

## C. 공유하기 (촬영/업로드)

섹션 제목: **"공유하기"**

### C-1. 촬영하기 버튼

```
┌────────────────────────────────────┐
│ ┌────┐                            │
│ │ 🎥 │  촬영하기                   │  녹색 그라데이션 배경
│ └────┘  실시간으로 스윙 영상을       │
│         촬영해 보세요               │
└────────────────────────────────────┘
```

| 항목 | 상세 |
|------|------|
| 스타일 | 녹색 그라데이션 배경 (golf-green → green-600), 흰색 텍스트 |
| 아이콘 | Camera (반투명 흰색 원형 배경 안) |
| 주 텍스트 | "촬영하기" |
| 부 텍스트 | "실시간으로 스윙 영상을 촬영해 보세요" |
| 탭 동작 | 기기 기본 카메라 호출 |

#### 촬영 플로우

```
[촬영하기 버튼 탭]
    │
 <input type="file" accept="video/*" capture="environment">
    │
 기기 카메라 앱 실행
    │
 ┌────┴────┐
촬영 완료   취소
 │          │
유효성 검사  아무 동작 없음
 │
 ┌──┴──┐
통과    실패 → 에러 토스트
 │
자동 업로드 (POST /api/media/upload)
 │
완료 → P3(미디어 선택)으로 이동 (해당 영상 자동 선택)
```

| 유효성 항목 | 제한 |
|------------|------|
| 포맷 | MP4, MOV, WEBM |
| 파일 크기 | 최대 500MB |
| 영상 길이 | 최대 10분 |
| 스토리지 | 잔여 용량 확인 (5GB 한도) |

### C-2. 영상 업로드 버튼

```
┌────────────────────────────────────┐
│ ┌────┐                            │
│ │ 📤 │  영상 업로드                │  흰색 배경, 테두리
│ └────┘  갤러리에서 영상을            │
│         업로드할 영상을 선택하세요    │
└────────────────────────────────────┘
```

| 항목 | 상세 |
|------|------|
| 스타일 | 흰색 배경, 회색 테두리, 호버 시 녹색 테두리 |
| 아이콘 | Upload (회색 원형 배경 안) |
| 주 텍스트 | "영상 업로드" |
| 부 텍스트 | "갤러리에서 영상을 업로드할 영상을 선택하세요" |
| 탭 동작 | 공유 모드 설정 후 P2(비율 선택)으로 이동 |

#### 업로드 플로우

```
[영상 업로드 버튼 탭]
    │
 isShareMode = true 설정
    │
 P2(비율 선택)으로 이동
    │
 (이후 P2 → P3에서 미디어 업로드 및 선택 진행)
```

- "영상 업로드" 버튼은 **파일 선택기를 직접 열지 않음**
- 공유 모드(isShareMode=true)를 설정하고 P2로 이동
- 실제 파일 업로드는 P3(미디어 선택) 화면에서 수행

## D. 빠른 시작

섹션 제목: **"빠른 시작"**

2열 그리드 레이아웃

### D-1. 새 프로젝트 시작

```
┌──────────────────┐
│                  │
│     ┌────┐      │
│     │ ⊕  │      │  녹색 배경 원형 아이콘
│     └────┘      │
│                  │
│  새 프로젝트 시작  │
│                  │
└──────────────────┘
  점선 테두리, 흰색 배경
```

| 항목 | 상세 |
|------|------|
| 스타일 | 흰색 배경, 점선 테두리, 호버 시 녹색 테두리 |
| 아이콘 | Plus (녹색, 녹색 반투명 원형 배경) |
| 텍스트 | "새 프로젝트 시작" |
| 탭 동작 | `isShareMode = false` 설정 → P2(비율 선택)으로 이동 |

### D-2. 최근 프로젝트 복제

```
┌──────────────────┐
│                  │
│     ┌────┐      │
│     │ 📋 │      │  파란 배경 원형 아이콘
│     └────┘      │
│                  │
│ 최근 프로젝트 복제 │
│                  │
└──────────────────┘
  점선 테두리, 흰색 배경
```

| 항목 | 상세 |
|------|------|
| 스타일 | 흰색 배경, 점선 테두리, 호버 시 녹색 테두리 |
| 아이콘 | Copy (파란색, 파란 반투명 원형 배경) |
| 텍스트 | "최근 프로젝트 복제" |
| 활성 조건 | 프로젝트가 1개 이상 있을 때만 활성 |
| 비활성 시 | 50% 투명도, 탭 불가 |
| 탭 동작 | 가장 최근 수정된 프로젝트를 복제 |

#### 복제 API

```
POST /api/projects/{id}/duplicate

Response 201:
{
  "id": "new-uuid",
  "name": "원본 이름 (복사)",
  ...
}
```

- `updatedAt` 기준 가장 최근 프로젝트 자동 선택
- 복제 완료 시 → "프로젝트가 복제되었습니다" (success 토스트)
- 프로젝트 목록에 즉시 반영

## F. 내 프로젝트

섹션 제목: **"내 프로젝트"**

### 프로젝트 목록 (2열 그리드)

프로젝트가 있을 때 2열 그리드로 표시합니다.

#### 프로젝트 카드

```
┌────────────────────────┐
│                        │
│  [썸네일 이미지]         │  가로세로비: 16:9 (aspect-video)
│                        │
│              ┌───┐     │
│              │0:45│     │  우측 하단: 재생 시간 뱃지
│              └───┘     │  (검정 반투명 배경)
├────────────────────────┤
│ 드라이버 샷 모음     [⋮]│  프로젝트명 (한 줄, 말줄임)
│ 🕐 2일 전              │  상대 시간
└────────────────────────┘
```

| 요소 | 상세 |
|------|------|
| 썸네일 | 프로젝트 대표 이미지. 없으면 Plus 아이콘 표시 |
| 재생 시간 뱃지 | `duration > 0`일 때만 표시, "M:SS" 형식 |
| 프로젝트명 | 한 줄, 넘치면 말줄임 (truncate) |
| 시간 정보 | Clock 아이콘 + 상대 시간 텍스트 |
| ⋮ 더보기 | 원형 버튼, 탭 시 하단 시트 열기 |

### 상대 시간 표시 규칙

| 조건 | 표시 |
|------|------|
| 1시간 미만 | "방금 전" |
| 1~23시간 | "N시간 전" |
| 24~47시간 | "어제" |
| 48시간 이상 | "N일 전" |

### 재생 시간 표시 형식

`formatDuration(초)` → "M:SS"

| 입력 | 출력 |
|------|------|
| 45초 | "0:45" |
| 120초 | "2:00" |
| 32초 | "0:32" |

### 카드 탭 동작

```
[프로젝트 카드 썸네일 영역 탭]
    │
 currentProject = 선택한 프로젝트 설정
    │
 P5(에디터)로 이동
```

### ⋮ 더보기 탭 동작

```
[⋮ 버튼 탭]
    │
 선택된 프로젝트 저장
    │
 하단 시트(Bottom Sheet) 열기
```

```
└────────────────────────────────────┘
  반투명 검정 배경 딤
```

| 항목 | 동작 |
|------|------|
| 등장 | 하단에서 슬라이드 업 (spring 애니메이션) |
| 배경 딤 | 반투명 검정, 탭 시 시트 닫힘 |
| 드래그 핸들 | 시트 상단 중앙의 작은 바 |

#### 메뉴 항목

| 메뉴 | 아이콘 | 동작 | 비고 |
|------|--------|------|------|
| 복제 | Copy | 프로젝트 복제 API 호출 → 목록에 추가 | - |
| 이름 변경 | Edit2 | 이름 변경 다이얼로그 열기 | - |
| 삭제 | Trash2 | 프로젝트 삭제 API 호출 → 목록에서 제거 | 빨간 텍스트 |

```
POST /api/projects/{id}/duplicate

Response 201:
{
  "id": "new-uuid",
  "name": "원본 이름 (복사)",
  ...
}
```

- 복제 완료 → "프로젝트가 복제되었습니다" (success 토스트)

#### 삭제

```
DELETE /api/projects/{id}

```

- 삭제 즉시 실행 (확인 다이얼로그 없음, 프로토타입 기준)

- **실 서비스**: 삭제 전 확인 다이얼로그 추가 권장 ("이 프로젝트를 삭제하시겠습니까? 이 작업은 되돌릴 수 없습니다.")

### 이름 변경 다이얼로그

하단 시트에서 "이름 변경" 탭 시 표시됩니다.

```
┌────────────────────────────────────┐
│                                    │
│     프로젝트 이름 변경              │
│                                    │
│  ┌──────────────────────────────┐ │
│  │ 드라이버 샷 모음              │ │  입력 필드 (기존 이름 프리필)
│  └──────────────────────────────┘ │  autofocus
│                                    │
│     [취소]          [변경]         │
│                                    │
└────────────────────────────────────┘
  반투명 검정 배경 딤 (더 진하게)
```

| 항목 | 규칙 |
|------|------|
| 등장 | 스케일 0.9→1 + 페이드 인 |
| 입력 필드 | 기존 프로젝트명이 미리 채워짐, autofocus |
| placeholder | "새 이름을 입력하세요" |
| 유효성 | 빈 문자열 불가 (trim 적용), 최대 50자 |
| [변경] 버튼 | 입력값이 비어있으면 비활성 |
| [취소] 버튼 | 다이얼로그 닫기 (변경 없음) |

#### 변경 API

```
PATCH /api/projects/{id}

Request: { "name": "새 이름" }
Response 200: { "id": "uuid", "updatedAt": "..." }
```

- 변경 완료 → "프로젝트 이름이 변경되었습니다" (success 토스트)
- 다이얼로그 자동 닫힘

## 네비게이션 동선 요약

```
P1 대시보드
 ├─ ← 뒤로가기 → 홈 화면
 ├─ 촬영하기 → 기기 카메라 → 업로드 → P3(미디어 선택)

 ├─ 템플릿 선택 → P2(비율 선택) [selectedTemplate 설정]
 ├─ 프로젝트 카드 탭 → P5(에디터) [currentProject 설정]
 ├─ 프로젝트 ⋮ → 하단 시트 (복제/이름변경/삭제)
 └─ 하단 네비게이션 → 각 탭 화면
```
