# GolfSnap 변경 로그

> **파일**: `_changelog.md`  
> **용도**: 명세서 변경 시 이 파일에 기록. Claude에게 해당 버전만 전달하면 변경된 섹션의 디스크립션만 업데이트.

---

## 변경 유형

| 유형 | 태그 | 설명 |
|------|------|------|
| 변경 | `### 변경` | 기존 기능의 동작/디자인/조건 변경 |
| 신규 | `### 신규` | 새로운 기능/요소 추가 |
| 삭제 | `### 삭제` | 기존 기능/요소 제거 |
| 확정 | `### 확정` | TBD 항목이 확정됨 |

## 항목 형식

```
- **[섹션 ID]**: 요약 (한 줄)
  - 이전: (변경/삭제 시) 기존 내용
  - 이후: (변경/신규 시) 변경된 내용
  - 사유: 왜 바꿨는지
```
