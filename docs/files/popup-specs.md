# 팝업/모달/다이얼로그 명세

> **파일**: `popup-specs.md`
> **최종 수정**: v1.0 (2026-02-11)

---

## 개요

앱 전체에서 사용되는 팝업, 모달, 바텀시트, 토스트의 통합 명세서입니다.

### 팝업 유형 분류

| 유형 | 표시 방식 | 해당 컴포넌트 |
|------|----------|-------------|
| 바텀시트 | 하단에서 슬라이드 업 | 프로젝트 메뉴, 속도, 볼륨, 전환효과 |
| 모달 다이얼로그 | 중앙 정렬 카드 | 이름 변경, 카메라 알림, 공유, 삭제 확인 |
| 풀스크린 패널 | 전체 화면 | 텍스트, 필터, 오디오, 스티커, AI 어시스턴트 |
| 토스트 | 하단 플로팅 | 토스트 알림 |

### 공통 애니메이션

| 유형 | 진입 | 퇴장 |
|------|------|------|
| 바텀시트 | y: 100% → 0 (spring) | y: 0 → 100% |
| 모달 다이얼로그 | scale 0.9→1, opacity 0→1 | scale 1→0.9, opacity 1→0 |
| 풀스크린 패널 | opacity 0→1, scale 0.95→1 | opacity 1→0, scale 1→0.95 |
| 토스트 | y: 30→0, scale 0.95→1, opacity 0→1 | y: 0→20, scale 1→0.95 |

## 1. P1 대시보드 팝업

### 1.1 프로젝트 메뉴 (바텀시트)

```
┌────────────────────────────────────┐
│  ┌───── 핸들 바 (w-12 h-1) ─────┐ │
│  └──────────────────────────────┘ │
│                                    │
│  📋 복제                           │
│  ✏️ 이름 변경                      │
│  🗑️ 삭제                          │  ← red-600
└────────────────────────────────────┘
```

| 항목 | 규칙 |
|------|------|
| 트리거 | 프로젝트 카드 ⋮ 버튼 탭 |
| 배경 딤 | bg-black/30, z-40 |
| 시트 배경 | bg-white, rounded-t-3xl, z-50 |
| 패딩 | px-4 py-6 |
| 핸들 바 | w-12 h-1, bg-gray-300, rounded-full, mx-auto, mb-6 |
| 항목 스타일 | flex, gap-3, px-4 py-3.5, rounded-xl |
| 항목 아이콘 | 20px (Copy, Edit2, Trash2) |
| 삭제 색상 | red-600 (아이콘 + 텍스트) |
| 항목 간격 | space-y-2 |
| 진입 애니메이션 | y: 100%→0, spring(damping 30, stiffness 300) |
| 딤 탭 | 시트 닫기 + selectedProject null |

#### 메뉴 동작

| 항목 | 동작 | 토스트 |
|------|------|--------|
| 복제 | 프로젝트 복제 | "프로젝트가 복제되었습니다" (success) |
| 이름 변경 | 바텀시트 닫기 → 이름 변경 다이얼로그 | — |
| 삭제 | 프로젝트 삭제 | "프로젝트가 삭제되었습니다" (success) |

## 2. P5 에디터 풀스크린 패널

모든 풀스크린 패널은 동일한 3분할 레이아웃을 사용합니다:

```
┌────────────────────────────────────┐
│  [제목]                         ✕  │  헤더 (고정)
├────────────────────────────────────┤
│                                    │
│  (스크롤 가능 콘텐츠)               │  flex-1
│                                    │
├────────────────────────────────────┤
│  [취소]          [적용/추가]        │  하단 버튼 (고정)
└────────────────────────────────────┘
```

| 공통 규칙 | 값 |
|----------|-----|
| 배경 | bg-gray-900/95 (딤) + bg-[#2c3441] (패널) |
| 크기 | w-full h-full (전체 화면) |
| Z-index | z-50 |
| 헤더 구분선 | border-b border-gray-700 |
| 하단 구분선 | border-t border-gray-700 |
| ✕ 닫기 | w-9 h-9, rounded-full, bg-gray-700/50 |
| ESC 키 | 패널 닫기 지원 |
| 취소 버튼 | bg-[#3d4554], white, font-semibold |
| 적용/추가 버튼 | bg-golf-green (기본) 또는 bg-pink-500 (스티커) |

---

### 2.1 텍스트 추가 패널

| 항목 | 규칙 |
|------|------|
| 트리거 | 에디터 툴바 "텍스트" 버튼 |
| 텍스트 입력 | textarea, h-28, autoFocus, bg-[#3d4554] |
| 탭 전환 | "스타일" / "애니메이션" (2탭) |

#### 스타일 탭

| 섹션 | 옵션 |
|------|------|
| 폰트 | 6종 (3열): Noto Sans, 나눔고딕, 나눔명조, Roboto, Montserrat, Playfair |
| 크기 | 슬라이더 16~72px, step 2 |
| 색상 | 10색 (5열): #FFF, #000, #F00, #0F0, #00F, #FF0, #F0F, #0FF, #FFA500, #800080 |

#### 애니메이션 탭

11종 (3열): 없음, 페이드 인, 페이드 아웃, 아래→위, 위→아래, 오른→왼, 왼→오른, 확대, 바운스, 타자기, 글로우

---

### 2.2 필터 패널

| 항목 | 규칙 |
|------|------|
| 트리거 | 에디터 툴바 "필터" 버튼 |
| 초기화 버튼 | 헤더 우측, bg-[#3d4554], text-sm |

#### 프리셋 (3열 그리드)

| 프리셋 | 밝기 | 대비 | 채도 | 색온도 |
|--------|------|------|------|--------|
| 없음 | 0 | 0 | 0 | 0 |
| 선명 | +10 | +20 | +30 | 0 |
| 부드러움 | +5 | -10 | -15 | +5 |
| 쿨톤 | 0 | +10 | +10 | -20 |
| 웜톤 | +5 | +5 | +15 | +25 |
| 프로 | +8 | +15 | +20 | +10 |

#### 세부 조정

4개 슬라이더 (range -50~+50, step 1): 밝기(Sun), 대비(Contrast), 채도(Droplets), 색온도(Thermometer)

## 3. P5 에디터 바텀시트

| 항목 | 규칙 |

|------|------|

| 프리셋 | 4개: 0.5x, 1x, 2x, 4x |

### 3.2 클립 볼륨

| 항목 | 규칙 |
|------|------|
| 트리거 | 에디터 툴바 "볼륨" (비디오 클립 선택 시) |
| 시트 배경 | bg-gray-800, rounded-t-3xl |

| 섹션 | 상세 |
|------|------|
| 볼륨 카드 | bg-gray-700 카드 내 슬라이더 + 음소거 토글 |
| 프리셋 | 5개: 0%, 25%, 50%, 75%, 100% |

| 값 변환 | UI: 0~100% ↔ 저장: 0~1 |

### 3.3 전환 효과

| 항목 | 규칙 |
|------|------|

| 효과 | 아이콘 | 설명 |

| 슬라이드 | ArrowRightLeft | 좌우 슬라이드 |
| 줌 | ZoomIn | 확대/축소 전환 |

## 4. 공통 팝업

### 4.1 토스트 알림

```
┌────────────────────────────────────┐
│  [아이콘]  메시지 텍스트            │
└────────────────────────────────────┘
```

| 항목 | 규칙 |
|------|------|
| 위치 | absolute bottom-24 left-4 right-4 |
| Z-index | z-[60] |
| 자동 사라짐 | 2.5초 후 |
| pointer-events | none (터치 통과) |
| 스타일 | rounded-xl, px-4 py-3, shadow-lg |

#### 타입별 스타일

| 타입 | 배경색 | 아이콘 |
|------|--------|--------|
| info | bg-gray-800 | Info |
| warning | bg-amber-600 | AlertTriangle |
| error | bg-red-600 | XCircle |
| success | bg-emerald-600 | CheckCircle |

---

### 4.2 공유 다이얼로그

2단계 다이얼로그 (입력 → 완료)

#### 입력 단계

| 항목 | 규칙 |
|------|------|
| Z-index | z-[100] |
| 제목 | "공유하기" (text-lg, bold) |
| 제목 입력 | maxLength 100, placeholder "제목을 입력하세요" |
| 내용 입력 | textarea rows=4, maxLength 500, 글자 수 카운터 |
| 공유 버튼 비활성 | 제목+내용 모두 비어있을 때 disabled:opacity-50 |

#### 완료 단계

| 항목 | 규칙 |
|------|------|
| 아이콘 | Check (32px) in 녹색 원형 (64px) |
| 제목 | "공유가 완료되었습니다" |
| 설명 | "콘텐츠가 성공적으로 공유되었습니다." |
| 확인 버튼 | bg-golf-green → onShare 콜백 → 닫기 |

---

### 4.3 클립 삭제 확인

```
┌────────────────────────────────────┐
│  클립 삭제                          │  text-lg, bold
│  선택한 클립을 삭제하시겠습니까?    │  text-sm, gray-600
│                                    │
│  [취소]            [삭제]           │  삭제: bg-red-600
└────────────────────────────────────┘
```

| 항목 | 규칙 |
|------|------|
| 트리거 | 에디터 툴바 "삭제" (클립 선택 상태) |
| 배경 딤 | bg-black/50, z-50 |
| 카드 | bg-white, rounded-2xl, p-6, max-w-sm |
| 취소 버튼 | bg-gray-100, gray-700 |
| 삭제 버튼 | bg-red-600, white |
